---
title: "November 2025 Release"
description: "Major SDK improvements, security fixes, and UX enhancements"
---

Comprehensive updates including OpenAI SDK middleware, Next.js 16 upgrade, IndexedDB migration, and critical security fixes.

## November 18, 2025

- **Multiple Prompt Prevention:** Fixed issue preventing users from submitting multiple prompts while AI response is being generated, improving chat reliability and preventing duplicate submissions ([#583](https://github.com/supermemoryai/supermemory/pull/583))

## November 15, 2025

- **Document Source Fix:** Resolved document source tracking issues to ensure proper attribution of uploaded files and imported content ([#580](https://github.com/supermemoryai/supermemory/pull/580))
- **Memory Dialog UX:** Fixed interaction bug where memories would open when clicking while delete memory dialog is active ([#582](https://github.com/supermemoryai/supermemory/pull/582))
- **Chat Deletion Confirmation:** Added inline confirmation dialog for chat deletion to prevent accidental data loss ([#581](https://github.com/supermemoryai/supermemory/pull/581))

## November 13, 2025

- **File Upload Source Tracking:** Enhanced source tracking for file uploads to maintain proper document metadata ([#579](https://github.com/supermemoryai/supermemory/pull/579))

## November 11, 2025

- **Organization Switching:** Fixed org switch issues on consumer platform when dev organization exists, ensuring smoother navigation between environments ([#577](https://github.com/supermemoryai/supermemory/pull/577))
- **Personal Assistant Cookbook:** Updated documentation with improved examples and guidance for building personal assistants with Supermemory ([#575](https://github.com/supermemoryai/supermemory/pull/575))
- **API Documentation Improvements:** Enhanced documentation for document/by-id endpoint with detailed warnings and usage guidelines based on community feedback ([#550](https://github.com/supermemoryai/supermemory/pull/550))
- **OpenAI Python SDK Middleware:** Added middleware functionality for Python SDK enabling automatic memory injection without AI SDK dependency ([#546](https://github.com/supermemoryai/supermemory/pull/546))

## November 10, 2025

- **Self-Hosting Documentation:** Added comprehensive self-hosting guide link to README for users wanting to run Supermemory on their own infrastructure ([#573](https://github.com/supermemoryai/supermemory/pull/573))
- **Past Due User Improvements:** Enhanced handling of past due and pending users with better error messages and recovery flows ([#572](https://github.com/supermemoryai/supermemory/pull/572))
- **Bookmark Import Optimization:** Fixed excessive icon refetch issue when importing large numbers of bookmarks, significantly improving import performance ([#496](https://github.com/supermemoryai/supermemory/pull/496))
- **WXT Storage API Migration:** Browser extension now uses WXT's storage API instead of raw Chrome storage API for improved reliability and consistency ([#566](https://github.com/supermemoryai/supermemory/pull/566))

## November 9, 2025

- **Sentry Error Resolution:** Fixed multiple production errors reported by Sentry (CONSUMER-APP-FF, CONSUMER-APP-1T, CONSUMER-APP-86, CONSUMER-APP-7H, CONSUMER-APP-4F, CONSUMER-APP-7X), improving overall application stability ([#570](https://github.com/supermemoryai/supermemory/pull/570))
- **PostHog Updates:** Updated analytics tracking based on PostHog findings to improve data quality ([#569](https://github.com/supermemoryai/supermemory/pull/569))
- **Hydration Fix:** Resolved React hydration issue affecting the model selector component ([#568](https://github.com/supermemoryai/supermemory/pull/568))

## November 7, 2025

- **OpenAI Response API Support:** Added support for OpenAI TypeScript SDK's responses API, expanding integration possibilities ([#549](https://github.com/supermemoryai/supermemory/pull/549))
- **Security Fix - Message Verification:** Fixed critical security vulnerability where messages could be sent to any website without verifying the recipient ([#565](https://github.com/supermemoryai/supermemory/pull/565))
- **IndexedDB Migration:** Migrated chat persistence from localStorage to IndexedDB to resolve QuotaExceededError and support larger chat histories ([#560](https://github.com/supermemoryai/supermemory/pull/560))

## November 5, 2025

- **Next.js 16 Upgrade:** Upgraded to Next.js 16 for improved performance, better developer experience, and access to latest features ([#557](https://github.com/supermemoryai/supermemory/pull/557))

## November 4, 2025

- **Google Drive Temporarily Disabled:** Google Drive connections temporarily disabled with clear UI indicators due to upstream provider issues. Users will see explanatory messages until the issue is resolved ([#559](https://github.com/supermemoryai/supermemory/pull/559))
- **Chrome Extension Update:** Updated Chrome extension package with latest dependencies and improvements ([#556](https://github.com/supermemoryai/supermemory/pull/556))

## November 3, 2025

- **Browser Extension Markdown Conversion:** Browser extension now converts HTML to markdown when saving entire web pages using TurndownService, making saved content more readable and portable ([#548](https://github.com/supermemoryai/supermemory/pull/548))
- **README 2.0:** Major README overhaul with light/dark mode compatible logo, badges for Web App/Chrome Extension/Raycast Extension, and detailed feature sections ([#547](https://github.com/supermemoryai/supermemory/pull/547))
- **MCP Connection Helper:** Added navigation to manual MCP setup documentation when users encounter connection issues ([#552](https://github.com/supermemoryai/supermemory/pull/552))

## November 2, 2025

- **Package Cleanup:** Removed unused dependencies to reduce bundle size and improve build performance ([#551](https://github.com/supermemoryai/supermemory/pull/551))

## October 29, 2025

- **Chat Message Scrolling:** Improved scroll behavior in chat interface for smoother user experience ([#544](https://github.com/supermemoryai/supermemory/pull/544))

- **Mobile Responsiveness:** Enhanced mobile chat interface with better skeleton loading states and responsive padding ([#541](https://github.com/supermemoryai/supermemory/pull/541))
- **Type Safety Improvements:** Added type safety annotations and fixed hook dependencies in chat components ([#521](https://github.com/supermemoryai/supermemory/pull/521))

## October 28, 2025

- **Streaming Memory Capture:** Added streaming support to @supermemory/tools middleware with improved memory handling for real-time assistant responses ([#539](https://github.com/supermemoryai/supermemory/pull/539))

- **Pricing Display Fix:** Fixed pricing information display in settings panel ([#528](https://github.com/supermemoryai/supermemory/pull/528))
